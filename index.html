@sham-button

Some basic button mixins to get you started.

These button mixins are designed to be very flexible. They take five variables $color1, $color2, $text-color, $font-size, and $radius. In some cases the colors are used for hover states and in others as parts of a gradient. Colors are darkened and lightened as needed for hover and active states. A little math is used on the $font-size variable to set an appropriate amount of padding. Add to the these by adding or changing styles to your button classes.

Installation

Naturally, you'll need Sass to use these mixins.

To install using Bower:

bower install sham-button

Or include the main .scss file in your project

@import 'sham-button';

Usage

Include something like this in your markup:

<div class="my-fancy-button">Testing</div>

Style it with something like this:

.my-fancy-button {@include sham-button-bordered (#EF382E, #e6e6e6, white, 14px, 30px);}

About Sham

We here at Identified use automative-themed names for our repos. "sham" – derived from chamois, a cloth comonly used to polish cars — is our all-purpose toolbox containing SASS mixins and variables to help us quickly build web apps.
